# SiteCheck — Website Health Auditor

One-command website audit: SSL, security headers, speed, SEO basics, redirects. Professional report in seconds.

```bash
sitecheck example.com
sitecheck example.com --json
```

Checks: SSL certs, security headers (HSTS, CSP, X-Frame), response time, redirect chains, SEO basics (title, meta, viewport).
